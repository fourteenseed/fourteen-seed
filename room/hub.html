gaining the qualification

Claude Certified Architect Foundations

CCAF (also CCA-F) is Anthropic's first architecture-level technical certification. It validates that you can make sound design decisions when building production applications with Claude: orchestration, tool integration, context management, structured output, and reliability under real conditions.

For us, CCAF matters because it gives the work a shared standard. The exam is rigorous enough that passing it is a meaningful signal to clients, collaborators, and employers. It maps well to what we're all building: agentic workflows, multi-step reasoning, and the practical question of how to make Claude reliably useful.

3

CCAF exam

120 minutes, 60 questions, closed-book. Five weighted domains. Passing score 720 out of 1,000. Cost: $99. Target: all cohort members through before end of July 2026.

The five exam domains

The exam draws from five weighted domains. The sequence below follows weighting, largest first.

01  ·  largest domain

Agentic architecture and orchestration

Designing multi-agent systems that work: Agent SDK, hub-and-spoke topologies, lifecycle hooks, and structuring orchestration so complex tasks complete reliably even when steps fail or branch.

27% of exam
02

Prompt engineering and structured output

Context engineering, JSON schemas, extraction patterns, few-shot techniques. Getting Claude to produce exactly what your system needs, in the format it requires, consistently.

20% of exam
03

Claude Code configuration and workflows

CLAUDE.md, agent skills, plan mode, slash commands, CI/CD integration. Configuring Claude Code for real development environments and wiring it into production workflows.

20% of exam
04

Tool design and MCP integration

Building MCP servers, defining tool boundaries, working with resources and prompts, and deciding when and how to expose capabilities to Claude.

18% of exam
05

Context management and reliability

Long-context strategies, multi-agent handoffs, error propagation, and escalation paths. Building systems that degrade gracefully and recover without losing work.

15% of exam

Exam format

Delivered online through Anthropic Academy via Skilljar. Closed-book, timed, no AI assistance.
